# **App Name**: CrediManage

## Core Features:

- Partner Management: Register, search, and manage partner information.
- Loan Management: Create, edit, and view active or historical loans, with customizable fields and association with partners.
- Installment Generation: Automatically generate and manage installments per loan.
- Payment Logging: Record individual or batch payments with smart partner search and duplicate payment validation.
- Receipt Generation: Automatically generate payment receipts upon marking an installment as paid.
- Financial Reporting: Visualize financial metrics by period with filters and export options.
- Anomaly Detection: Employ an anomaly detection tool that flags any inconsistencies (cuotas pagadas fuera de orden, montos incoherentes) in historical data, allowing for thorough validation during reconstruction.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and security in financial matters.
- Background color: Light gray (#F5F5F5), nearly desaturated from the primary hue to offer a neutral backdrop.
- Accent color: A vibrant purple (#9C27B0), to draw attention to key actions and elements.
- Body font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look, suitable for both headlines and body text.
- Use large, professional icons to label modules and actions, improving usability, and aiding navigation.
- Design a 100% responsive layout that adapts to different screen sizes, ensuring optimal mobile accessibility.
- Incorporate subtle transitions and animations to enhance user experience, such as loading screens or feedback on user interactions.
